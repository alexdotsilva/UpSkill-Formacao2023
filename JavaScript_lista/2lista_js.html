EX. 01 - Explique a diferença entre os operadores de comparação ==, === e !=.

== - indica igualdade de valor. Por exemplo se indicar x=1. x==1 e x=="1" vão retornar como true. Contanto que o valor indicado seja 1, não importa a notação usada (binario, hexadecimal, string ou real).

=== - idica igualdade de valor e de tipo. Diferente do exemplo acima, a comparação só retorna true se o valor e o tipo forem iguais, assim x==="1" retorna falso.

!= - indica não igualdade, ou se é diferente. Assim, x!=2 ou x!="2" retorna true. Da mesma maneira podemos usar !=== para indicar não deiferença de valor ou tipo.

EX. 05 - Quais são os diferentes tipos de loops que podem ser utilizados em JavaScript?

for - O loop "para" corre uma função por um número de vezes definido em um bloco de código. Sendo preciso definir uma variavel contadora, a função que deve ser usada e um incremento para o número de vezes (for (let i = 0; i < 5; i++){retorno}). Ainda podem ser usados os comandos for/in e for/of para interações com as propriedades de um objeto e valores de um objeto, nessa ordem.

while - O loop "enquanto" interage com um bloco de código enquanto a condição apresentada for verdadeira (while (x==1){retorno}).

do while - Do mesmo modo, o loop "faça enquanto", interage com o bloco de código uma primeira vez antes de verificar se a condição é verdadeira. Assim, se a condição mudar antes da verificação o programa sai do loop (do {código a correr} while (x!=0)).

EX. 08 - Elabore uma pesquisa para descrever como funciona o sistema de módulos no JavaScript e como importá-los no código.

Módulos são uma maneira de compartimentar o codigo de maneira a facilitar a manutenção e o possível re-uso ou remoção do mesmo.
O padrão Module é uma forma de imitar o conceito de classes, visto que JavaScript não da suporte ao conceito de classes de modo nativo, permitindo armazenar métodos e variaveis de maneira pública ou privada.
Assim, para que as funções/variaveis/contantes possam ser usadas em outros modulos as keywords "import" e "export" são usadas.
Para importar módulos em JavaScript, utiliza-se a sintaxe import seguida do nome da variável ou função desejada, juntamente com o caminho (relativo ou absoluto) do arquivo do módulo. Isso oferece flexibilidade na organização do código e, evita conflitos de nomeação global. Além de permitir que os módulos possam ser carregados de forma assíncrona, melhorando o desempenho em determinadas situações.
